rat.generators.dispatch¶

Utilies for reading events from a dispatcher stream.

rat.generators.rootfile¶

Utilities for reading from TTrees in ROOT files.

rat.generators.rootfile.airfill_dsreader(filename)[source]¶

Read events from an air-fill style packed ROOT file

Returns an iterator over the PackedEvent objects in the PackEv tree in file filename.

rat.generators.rootfile.dsreader(filename)[source]¶

Read events from a RAT ROOT file

Returns an iterator over the DS objects in the T tree in file filename.

rat.generators.rootfile.packed_dsreader(filename)[source]¶

Read events from a packed ROOT file

Returns an iterator over the PackedRec objects in the PackT tree in file filename.

rat.generators.rootfile.tree_reader(filename, tree, branch, obj)[source]¶

Read data from the branch named branch of a ROOT tree named tree in a file filename. obj is a “reference” to an object of the type stored in the tree (cf. SetBranchAddress(branch, &obj)).
