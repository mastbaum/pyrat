rat.core.processor¶

Processor-related classes.

class processor.Processor(name)[source]¶

A processor represents a chunk of analysis code that can be placed in the event loop.

Parameters:name – The name of the processor
end_run()[source]¶

Called once at the end of a run.

event(event)[source]¶

Called for every event.

Parameters:ev – The event
start_run()[source]¶

Called once at the start of a run.

exception processor.ProcessorAbort(value)[source]¶

When raised by a processor, processing of this event is immediately terminated and no further processors are run on this event.

class processor.ProcessorBlock[source]¶

A processor block is a list of processors which are each called for every event.

end_run()[source]¶

Call the end_run methods for all processors in the block. Called at the end of a run.

event(ev)[source]¶

Call the event methods of all processors in the block. Called once per event.

Parameters:ev – The event
start_run()[source]¶

Call the start_run methods for all processors in the block. Called at the start of a run.

exception processor.ProcessorFail(value)[source]¶

When raised by a processor, signifies that the processor task failed, but later processors are still executed.

class ratdb.DB[source]¶

RATDB is the database of constants used by RAT for all adjustable parameters, including physical properties of materials, detector geometry, calibration constants, lookup tables, and control parameters that specify how processing should be done.

Parameters:filename – Name of file to read table from

Catch UNIX signal sende to the main application by the operating system.

class signal_handler.SignalHandler[source]¶

Handle SIGINT gracefully: the first time sets a flags, giving processes time to clean up, while the second time terminates immediately.

handler(signum, frame)[source]¶

Handler function called by signal on the first SIGINT.

is_term_requested()[source]¶

Returns true if termination has been requested by a SIGINT signal.

sigint()[source]¶

Called by handler. If this is the first Ctrl-C, set the signal pending flag and set future `SIGINT`s to actually terminate the program.
